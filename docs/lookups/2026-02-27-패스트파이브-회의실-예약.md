# 패스트파이브 회의실 예약을 디스코드에서 자연어로 처리하기

## 문제 정의
패스트파이브 신논현 2호점 회의실 예약을 할 때마다 웹(members.fastfive.co.kr)에 로그인해서 수동으로 처리해야 한다. 디스코드에서 자연어로 예약하고, 이미 예약된 시간이면 피드백을 받고 싶다.

## 분석
- 사람 개입: 매번 필요 — 사람이 직접 예약 의사를 전달해야 함
- 반복 주기: 비정기적 — 필요할 때만 발생
- AI 추론: 가벼운 수준 — 자연어를 날짜/시간/회의실로 파싱
- 외부 서비스: 3개 이상 — LLM(Claude Max API Proxy), 패스트파이브 웹(로그인+스크래핑), 디스코드, Whisper(음성 입력 가능성)

## 추천
**n8n 워크플로우**

디스코드 입력 → LLM으로 자연어 해석 → 패스트파이브 웹 예약 → 결과 피드백이라는 멀티서비스 연동 파이프라인이 핵심이며, 이벤트 기반(디스코드 메시지)으로 트리거되어야 하므로 n8n이 가장 적합하다.

### 다른 방식이 안 되는 이유
- **launchd + 셸 스크립트**: 디스코드 트리거 수신, LLM 연동, 웹 스크래핑을 셸 스크립트로 조합하기엔 복잡도가 과도하고 유지보수가 어려움
- **Apple Shortcuts**: 디스코드 메시지 트리거를 받을 수 없고, 웹 스크래핑/로그인 세션 관리가 불가능
- **Raycast 확장**: 키보드 즉석 호출 방식이라 디스코드에서 자연어로 예약한다는 요구사항과 맞지 않음
- **Claude Code 스킬**: 깊은 AI 추론이 아닌 자연어 파싱 수준이고, 디스코드 메시지 → 자동 처리 파이프라인이 필요하므로 워크플로우가 적합

### 구현 시 참고 사항
- 패스트파이브 웹 예약 시 로그인 세션 관리 필요 (쿠키/토큰 유지)
- 회의실 가용 여부 조회 후 예약 가능 여부 피드백
- 예약 페이지 URL: https://members.fastfive.co.kr/reserve/7
- 30분 단위 예약
