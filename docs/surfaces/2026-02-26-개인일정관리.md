# 개인 일정을 캘린더에 등록하는 게 귀찮아서 기억에 의존하고, 그로 인해 무의식적으로 주의가 분산된다

## 발견 과정
일/업무 영역 탐색에서 시작. 사용자가 개인적인 일정 이벤트들을 쉬운 채널로 등록하고, 적절한 때에 알림을 받고 싶다는 불편을 표현했다. 캘린더 앱에 등록하지 않는 이유를 물었을 때 "귀찮음"이라고 답했고, 기억에 의존하면서 무의식에서 기억하려는 시도가 주의를 뺏는다는 점이 핵심 고통으로 드러났다.

## 문제 정의
- 영역: 일/업무 (개인 일정 관리)
- 빈도: 가끔 / 비정기적
- 고통 정도: 짜증나는 정도 — 관리가 안 되고 무의식에서 기억하려는 시도로 주의가 뺏김
- 현재 대처: 기억에 의존, 캘린더 앱에 등록하지 않음
- 배경: 캘린더 앱에 등록하는 과정 자체가 귀찮아서 등록을 안 하게 됨. 더 쉬운 채널로 일정을 등록하고, 적절한 때에 알림을 받고 싶음
