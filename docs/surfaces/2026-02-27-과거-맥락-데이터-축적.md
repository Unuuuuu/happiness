# 맥락 수집 시스템에 과거 데이터가 없어 /work-scan이 유의미한 분석을 할 수 없다

## 발견 과정

/work-scan으로 업무 맥락을 스캔했으나, 맥락 수집 시스템이 오늘 가동되어 14일치 데이터가 테스트 이슈 2건과 GitHub Push 1건뿐이었다. 사용자가 "이전 데이터들이 축적되게끔 할 수 없을까" 포인트를 선택하여 /surface로 진행.

## 문제 정의

- 영역: 일/업무
- 빈도: 가끔 / 비정기적
- 고통 정도: 실질적 손해 — 데이터 없이 스캔하면 의미 없는 결과가 나와 판단 오류 발생
- 현재 대처: 무시하고 넘어감 — 데이터가 쌓일 때까지 기다리는 중
- 배경: 맥락 수집 시스템(Linear/GitHub 웹훅 → PostgreSQL)이 오늘 가동됨. 웹훅은 앞으로의 이벤트만 수신하므로, 시스템 가동 이전의 Linear 이슈 상태와 GitHub 활동은 수집되지 않은 상태.
