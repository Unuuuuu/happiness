# 패스트파이브 회의실 예약을 할 때마다 웹에 로그인해서 수동으로 처리해야 한다

## 발견 과정
일/업무 영역 탐색에서 사용자가 패스트파이브 공유오피스 회의실 예약의 불편함을 언급. 현재 웹(members.fastfive.co.kr)에 매번 로그인해서 수동 예약하고 있으며, 디스코드에서 자연어로 예약하고 싶다는 니즈를 확인.

## 문제 정의
- 영역: 일/업무
- 빈도: 가끔 / 비정기적
- 고통 정도: 짜증나는 정도
- 현재 대처: 매번 웹에 로그인해서 수동으로 예약
- 배경: 패스트파이브 신논현 2호점 이용 중. 30분 단위로 회의실 예약 가능. 디스코드에서 자연어로 예약하고 싶고, 이미 예약된 시간이면 안 된다는 피드백을 받고 싶음. 예약 페이지 URL은 https://members.fastfive.co.kr/reserve/7
