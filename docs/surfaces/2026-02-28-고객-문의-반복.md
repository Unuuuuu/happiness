# 동일 유형의 고객 문의가 반복 등장하여 매주 수동 처리에 시간과 대응 지연이 발생한다

## 발견 과정

/work-scan 180일 데이터 분석에서 "고객 문의 반복 등장" 패턴이 탐지되었다. 프로필 수정, 닉네임 변경, 식단 정보, 프로모션 날짜 등 비슷한 유형의 고객 문의가 Linear의 Done과 Todo 상태에 계속 반복 등장하는 것을 확인하고, 사용자가 이 포인트를 선택하여 문제를 구체화했다.

## 문제 정의

- 영역: 일/업무
- 빈도: 매주
- 고통 정도: 의미 있는 시간 낭비 + 실질적 손해 (대응 지연으로 인한 고객 불만족, 업무 병목)
- 현재 대처: 매번 수동으로 처리
- 배경: 180일간 같은 종류의 고객 문의가 Done 처리되어도 새로운 이슈로 계속 재등장한다. 각 문의를 매주 개별적으로 수동 처리하느라 다른 업무 시간이 줄어들고, 대응이 지연되면 고객 불만족으로 이어진다.
